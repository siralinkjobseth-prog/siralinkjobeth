SiraLink

Smart Recruitment, Better Future

Overview

SiraLink is a modern recruitment platform designed to connect job seekers with employers across Ethiopia. The platform provides job listings, profile management, notifications, analytics, and Telegram integration.

Features

- Telegram Authentication
- Job Search & Filtering
- Save Jobs
- Job Applications
- User Profiles
- Real-Time Notifications
- Admin Dashboard
- Analytics & Reports
- Supabase Backend
- Telegram Bot Integration

Project Structure

- Frontend (HTML, CSS, JavaScript)
- Supabase Database
- Supabase Edge Functions
- Telegram Bot
- Admin Management System

Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Supabase
- Telegram Bot API
