SiraLink Notification Engine

Overview

The Notification Engine is responsible for delivering important updates and alerts to users across the SiraLink platform.

Notifications are generated automatically based on user actions, system events, job updates, and administrator broadcasts.

The system supports:

- In-App Notifications
- Telegram Notifications
- Broadcast Messages
- Automated Alerts

---

Notification Goals

The notification system helps users:

- Discover new jobs
- Track applications
- Receive platform updates
- Get important announcements
- Stay engaged with opportunities

---

Notification Types

Job Notifications

Generated when:

- New job is posted
- Job matches user profile
- Job deadline is approaching

Examples:

- New Software Engineer position available
- New remote job matches your profile
- Application deadline ends tomorrow

---

Application Notifications

Generated when:

- Application submitted
- Application reviewed
- Application shortlisted
- Application accepted
- Application rejected

Examples:

- Application submitted successfully
- Your application has been shortlisted

---

System Notifications

Generated when:

- Profile updated
- Password changed
- Platform maintenance
- New feature released

Examples:

- Profile updated successfully
- New platform features available

---

Broadcast Notifications

Generated by administrators.

Examples:

- Recruitment campaign announcement
- Important company update
- General platform announcement

Notification Retention

90 Days

Read:
180 Days

- Send broadcasts
- Monitor delivery
